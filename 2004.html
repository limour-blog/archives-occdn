Docker：安装frp服务端与客户端

来源

https://hub.docker.com/r/snowdreamtech/frps

https://github.com/fatedier/frp

服务端部署

  • mkdir frps && cd frps && mkdir frps_log
  • nano docker-compose.yml
  • nano frps.ini
  • docker-compose up -d
  • tail frps_log/frps.log

客户端部署

  • mkdir frpc_001 && cd frpc_001
  • nano docker-compose.yml
  • nano frpc.ini
  • sudo docker-compose up -d
