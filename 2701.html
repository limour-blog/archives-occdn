搭建联通大王卡免流服务器

找一些免流的混淆

  • 可以用此项目扫描，确保混淆指向了CDN
  • 支持 TLSv1.3 与 H2，域名非跳转

这里随便找的微信公众号的图片CDN，mmbiz.qpic.cn，具体地址点此

创建Docker镜像

  • mkdir -p ~/app/dawang && cd ~/app/dawang && nano Dockerfile && nano docker-compose.yml
  • 安装解压软件：sudo apt install unzip
  • 下载免流的服务端：wget https://ghproxy.com/https://github.com/XTLS/Xray-core/releases/download/v1.8.1/Xray-linux-64.zip -O tmp.zip
  • unzip tmp.zip && rm tmp.zip && chmod +x xray
  • sudo docker build -t limour/dawang .
  • sudo docker run –rm -it limour/dawang -version

运行服务端

  • ./xray uuid 获得id
  • ./xray x25519 获得公钥和私钥
  • nano docker-compose.yml && nano ./config.json
  • sudo docker-compose up -d
  • sudo docker-compose logs
  • 访问8443端口，看看证书是不是对的

运行客户端

  • 下载测试用的客户端
  • 在程序目录下创建config.json，内容如上
  • 运行powershell脚本
  • 访问测试网址，判断是否生效
